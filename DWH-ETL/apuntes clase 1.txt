- Definición API?

Una API es un conjunto de definiciones y protocolos que se utiliza para desarrollar e integrar el software 
de las aplicaciones. API significa interfaz de programación de aplicaciones. Las API permiten 
que sus productos y servicios se comuniquen con otros, sin necesidad de saber cómo están implementados.
