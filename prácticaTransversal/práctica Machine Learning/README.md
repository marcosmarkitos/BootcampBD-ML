PRÁCTICA MACHINE LEARNING

Consiste en los siguientes ficheros:

- project: Directorio que incluye toda la información de la práctica.
	- img: Directorio donde se guardan las imágenes que se usan en el notebook para explicar mejor lo realizado en la práctica.
	- airbnb-listings-extract.csv : fichero con la información original (dataset de entrada).
	- Marcos Gutiérrez Práctica ML.ipynb : notebook con la práctica completa.
	- Práctica ML.pdf: enunciado de la práctica
	- test.csv : fichero con el conjunto de test (se genera de forma autómatica al ejecutar la práctica).
	- train.csv : fichero con el conjunto de train (se genera de forma autómatica al ejecutar la práctica).
